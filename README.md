3 Digit bulls and cows game for Hubot

휴봇 숫자야구 게임 모듈입니다


*명령어*

게임시작: '야구게임 시작'
순위보기: '야구게임 랭킹' 또는 '야구게임 순위'


*작동방법*

- 슬랙 채널에서 '야구게임 시작'을 치면 게임이 시작됩니다.
- 3개의 연속된 숫자를 입력하면 회차가 카운팅되고 결과가 보여집니다
ex) 
사용자입력:
123

봇 대답:
1회 [123] - 1S 1B

- 총 9회까지 진행됩니다
